# Assemblytics Galaxy Wrapper

This Galaxy wrapper for Assemblytics was created by Michael Schatz.

Assemblytics is a tool for detecting and analyzing structural variants by comparing a de novo genome assembly to a reference genome. It uses unique-anchor filtering to improve robustness in repetitive regions and reports multiple classes of structural variation along with summary tables and visualizations.

## Citation

Nattestad M, Schatz MC. Assemblytics: a web analytics tool for the detection of variants from an assembly. Bioinformatics. 2016;32(19):3021-3023. doi:10.1093/bioinformatics/btw369
